| U2  | Нужно придерживаться системы дизайна для приложений, которая принята в компании. Пользователи при работе должны видеть привычные цвета и брендированные элементы.                                          |                                                                                                                                       |
| U3  | На сайте и в интернет-банке клиент видит предложение открыть кредит с актуальными условиями                                                                                                                |                                                                                                                                       |

| R2  | Предусмотреть равномерное горизонтальное масштабирование и распределение запросов между серверами, приложениями и ЦОД с учетом, что АБС может масштабироваться только вертикально из-за своей базы данных. |                                                                                                                                       |
| R3  | Есть ожидания, что заявки на кредит по предодобренным предложениям будут обрабатываться сотрудниками в течение того же рабочего дня                                                                        | Сейчас заявки попадают в Кредитный конвейер раз в сутки из АБС. При этом ускорить обмен между базами и проводить его чаще невозможно. |

| P2  | Система должна обрабатывать заявки на депозит и на кредит в реальном времени                                                                                                                               | Необходимо, чтобы клиенту сразу же выдавалась информация об условиях, чтобы он не успел передумать                                    |

| S2  | Система должна обеспечивать возможность мониторинга состояния всех интеграций в реальном времени                                                                                                           | Необходимо для быстрого реагирования на инциденты                                                                                     |
| S3  | Система должна поддерживать версионирование API для обратной совместимости                                                                                                                                 | Для постепенного внедрения новой системы                                                                                              |
